# 급식 정보 (Lunch Info)

이 프로젝트는 NEIS(나이스) 교육행정정보시스템 API를 활용하여 전국 학교의 급식 정보를 조회하고 보여주는 웹 애플리케이션입니다.

## 주요 기능

*   **학교 검색**: 학교 이름으로 검색하여 급식 정보를 조회할 수 있습니다.
    *   **오프라인 검색**: `schools.json` 및 로컬 캐시를 기반으로 빠르게 검색합니다.
    *   **온라인 검색**: 오프라인 결과가 없을 경우 NEIS API를 통해 실시간으로 학교를 검색합니다.
*   **급식 정보 표시**: 조식, 중식, 석식 메뉴를 깔끔한 카드 형태로 제공합니다.
*   **개발자 모드**:
    *   교육청 코드 및 학교 코드 직접 입력 가능.
    *   JSON 파일 업로드 또는 API 응답 텍스트 붙여넣기를 통한 데이터 로드 지원.
*   **프리미엄 UI**: 모던하고 깔끔한 디자인(Inter 폰트, 반응형 레이아웃)을 적용했습니다.

## 파일 구조

*   `index_static.html`: 프론트엔드 진입점 (HTML 구조).
*   `style.css`: 전체 스타일시트 (프리미엄 디자인 적용).
*   `script.js`: 주요 로직 (학교 검색, API 호출, 데이터 파싱 등).
*   `fetch_schools.py`: (옵션) 전국 학교 정보를 가져와 `schools.json`을 생성하는 파이썬 스크립트.
*   `DB/`: (옵션) 날짜별 급식 데이터가 캐싱되는 디렉토리.

## 사용 방법

1.  `index_static.html` 파일을 웹 브라우저로 엽니다.
2.  **일반 모드**:
    *   학교 이름을 검색창에 입력합니다.
    *   검색 결과에서 학교를 선택합니다.
    *   날짜(YYYYMMDD)를 확인하고 "NEIS 불러오기" 버튼을 클릭합니다.
3.  **개발자 모드**:
    *   우측 상단의 "개발자 모드" 버튼을 클릭합니다.
    *   교육청 코드, 학교 코드, 날짜를 직접 입력하거나, JSON 파일을 업로드하여 테스트할 수 있습니다.

## 설정

*   **API 키**: `script.js` 파일 상단의 `OPENAPI_KEY` 변수에 본인의 NEIS Open API 키를 입력하면 더 안정적으로 사용할 수 있습니다. (기본 키가 포함되어 있으나, 트래픽 제한이 있을 수 있습니다.)
